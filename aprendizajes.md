# ¿Qué es Git y para qué sirve?

Git es un sistema de control de versiones distribuido que permite registrar los cambios realizados en los archivos de un proyecto a lo largo del tiempo. Sirve para trabajar de forma organizada, permitiendo volver a versiones anteriores y facilitar el trabajo colaborativo sin perder información.
